# My_Portfolio:
This is my personal portfolio built to showcase my skills and featured projects.

## TECH STACK:
+ React
+ TypeScript
+ Tailwind CSS

# React :
+ A JavScript Library used for creating Single-Page-Applications (SPA).
+ Unlike Javascript, react creates a Virtual DOM through which the changes happens to a specific element.
+ React Allows to create Reusable UI components.

# TypeScript:
+ TypeScript is an open-source programming language developed by Microsoft.
+ It is a statically typed superset of JavaScript.
+ It helps developers to catch errors during development rather than at runtime.

# Tailwind CSS:
+ A utility based CSS framework that enables developers to build custom user interfaces rapidly by using pre-defined, single-purpose utility classes directly within their HTML markup.
